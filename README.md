# 두루미 기록

## 프로젝트 시작

```bash
npm install
hugo server
```

## UI

- https://tailblocks.cc/ html block을 붙이면서 가볍게 스타일링합니다. 
- UI component 규칙을 적용해  basic / app / marketing 폴더에 섹션들을 붙여봅니다.

## Data/Record

- 엑셀에 접근해서 몇 가지 정보를 수정합니다. 변경된 정보를 반영하면서 휴고를 빌드합니다. 

```bash
hugo server --disableFastRender
```
